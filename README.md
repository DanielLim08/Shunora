# Shunora:
a multipurpose pomodoro study timer that centralizes many tools students use for time management. Has both an extension and a website.

# Problem we solve:
Students waste a lot of time procrastinating. This web app enhances student focus.

# Main Features:
 - customizable Pomodoro timers
 - focus mode with website blocker
 - calendar
 - reminders
 - to do list
 - resource organized hub
 - note taker
 - community leaderboard

# languages and frameworks:
javascript
HTML
CSS
